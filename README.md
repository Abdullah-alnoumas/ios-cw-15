# الهيكل Struct

### كيف تقوم بإنشاء الهيكل Struct جديد

```
struct NameOfStruct {
   	// our variables and Constants
}
```

### كيف نقوم باستعمال هذا الهيكل 

```
NameOfStruct(nameOfvariable: value)
```


### مثال:


```
struct Person{
    let name: String
    let age : Int
}
Person(name: "Adnan", age: 47)
```
